# flotsam: Local Tangent Space Alignment

`flotsam` implements Local Tangent Space Alignment (LTSA) for nonlinear
dimensionality reduction.

## Details

The main entry point is
[`ltsa()`](https://jlmelville.github.io/flotsam/reference/ltsa.md). It
accepts numeric matrices and data frames with observations in rows.
